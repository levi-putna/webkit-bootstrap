About Webkit Bootstrap

Webkit Bootstrap is a small template to get you started building desktop applications using Java, HTML, CSS and javaScript.

About

Welcome to the Webkit Bootstrap, this is a little template to get you started building desktop applications using Java, HTML, CSS and javaScript.

Content and UI layout for this About Window has been created using HTML and CSS. You can see the source for this page at www/html/about.htm, This container uses webkit so any HTML5 CSS3 functionality is supported.

Source

The source is available from GitHub (https://github.com/levi-putna/webkit-bootstrap), to contribute to the template please make a pull request.
